Introdução

O poder de compra se refere à capacidade de um indivíduo ou grupo adquirir bens e serviços com determinada quantia de dinheiro. Ao longo dos anos, essa capacidade tem sido diretamente influenciada por fatores como inflação, políticas econômicas e mudanças nos salários. O principal problema relacionado ao poder de compra é seus gastos em períodos de inflação elevada, quando os preços sobem mais rapidamente do que os salários. Em diversos momentos da história, a população viu seu poder aquisitivo diminuir por diversos fatores, como guerras, epidemias mundiais... E isso impacta o acesso a bens básicos e serviços, reduzindo o padrão de vida. Esta pesquisa tem como objetivo comparar o poder de compra em diferentes décadas, avaliando como as políticas econômicas e as flutuações nos preços influenciaram o cotidiano das pessoas.

Materiais e métodos

HTML E CSS (linguagem de programação usada para criação do código) Visual Studio Code (plataforma de edição de código) O HTML é uma linguagem padrão de programação, usada pra criar e estruturar paginas da web. Ele define a estrutura e os conteúdos da página, usando tags para indicar elementos diferentes, como os parágrafos, títulos e etc. Essa linguagem é base de qualquer página, assim como a usada nesse trabalho. O CSS é uma linguagem utilizada principalmente na estilização e formatação de páginas programadas em HTML. Com ele é possível mexer em cores, fontes, margem e outros elementos gráficos da página. Essa linguagem utilizada com o HTML é fundamental em muitos sites, assim como nesse trabalho. O visual Studio Code é um editor de código-fonte leve utilizado na edição e depuração de códigos de várias linguagens interagindo com elementos como sintaxe, extensões, ferramentas de depuração e outros. Esse programa desenvolvido pela Microsoft foi utilizado nesse trabalho pra editar os códigos HTML.

Conclusão

A inflação ao longo dos anos tem sido um dos fatores mais influentes na redução do poder de compra da população. Como visto, em períodos de inflação elevada, a capacidade de adquirir bens e serviços essenciais diminui drasticamente.
